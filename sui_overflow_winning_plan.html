Understanding the Sui Overflow Guidelines

Sui Overflow 2026 is Sui's flagship global virtual hackathon running from May 6, 2026 to June 20, 2026. Mysten Labs has structured this year's hackathon into high-impact categories aimed at expanding real-world usage, and decentralized storage scaling. Below is our visual assessment of the core and specialized tracks.

Core Hackathon Tracks

These tracks are broad and open, offering some of the largest project counts and ecosystem integration opportunities:

  • The Agentic Web (AI): Building autonomous AI agents that can act, trade, and orchestrate complex workflows using Sui's object model and parallelized transaction architecture.
  • DeFi & Payments: Creating fast, seamless financial primitives, yield products, and payment rails designed for high transaction volume.
  • Infra & DevX: Tooling, SDKs, Zero-Knowledge proofs, developer tooling, or infrastructure enhancements that simplify the Sui Move workflow.

Specialized Partner Tracks

Focusing on highly specific Mysten Labs technologies. Projects built here are highly valued because they leverage cutting-edge devnet/testnet layers:

  • Walrus Protocol: Mysten Labs' brand new decentralized storage and data availability network. Projects must use Walrus for storing and validating heavy off-chain files (blobs) such as NFTs, data sets, or decentralized website frontends (Walrus Sites).
  • DeepBook (v3): Sui's highly-optimized fully on-chain central limit order book (CLOB). Builders can utilize flash loans, custom balance managers, dynamic maker/taker fee rebates, and orderbook mechanics to power advanced trading platforms.

The Winning Strategy Checklist

Sui Foundation judges score submissions based on: (1) Technical complexity (Move smart contracts + clean frontend integrations), (2) Ecosystem fit and unique utilization of Sui's Object Model, (3) Practical utility and commercial readiness, and (4) Visual excellence of the interface. Simple proof-of-concept demos fail. Complete, production-grade applications win.

Aether.ai — Decentralized AI Hedging Fund

Estimated Score: 96/100

Aether.ai solves a massive dilemma in Web3 DeFi: the lack of active portfolio hedging for passive users. Instead of passive liquidity pools that suffer impermanent loss, Aether.ai deploys an active, autonomous trading agent that monitors volatility and executes hedges on DeepBook v3, while maintaining absolute security on-chain and verifiable audit logs on Walrus.

System Architecture & Composability

Aether.ai is split into three core layers designed to leverage the best of Sui's infrastructure:

1. On-Chain Smart Vaults (Sui Move)

  • Users deposit funds into a shared `SmartVault` object.
  • The vault tracks the deposits, withdrawals, and authorizes a specific `AgentPubKey` to perform trades.
  • Drawdown Safeguard: The Move contract enforces that the AI agent cannot withdraw funds to external addresses, and cannot trade assets outside an approved whitelist. It also enforces daily trade limits and loss thresholds directly at the smart contract level.

2. AI Decision Engine & Oracle Feed

  • An off-chain agent script (built in Node.js/TypeScript using Sui Agent Kit) constantly monitors oracle feeds, market pricing, and historical indicators.
  • When the agent decides to execute a trade, it wraps the trade instructions inside a Sui Programmable Transaction Block (PTB) to execute atomically.
  • The agent can also query DeepBook v3 for Flash Loans to execute atomic market arbitrage.

Verifiable Decision Ledger on Walrus

Walrus Integration for Unprecedented Trust

Most AI trading systems are black boxes. Aether.ai forces transparency: the agent compiles its analysis (volatility indices, LLM sentiment, market signals, confidence score) into a structured JSON file. It uploads this to **Walrus Devnet** as a blob. The Move contract's `execute_trade` function requires the agent to pass the `WalrusBlobID`. The contract then mints a non-transferable on-chain `TradeProof` object, linking the exact on-chain tx to the decision blob. Anyone can read the agent's logic for every transaction!

WalrusVault — Cryptographic File Space

Estimated Score: 89/100

WalrusVault addresses the massive enterprise security challenge in Web3. While platforms like Google Drive or Notion are highly centralized, standard blockchains cannot cost-effectively store big enterprise files (PDFs, architectural blueprints, medical records). WalrusVault leverages the raw file hosting power of Walrus Protocol, pairing it with Sui smart contract-based access control rules to make an extremely secure, private, decentralized workspace.

How it Works

  • Zero-Knowledge Client-Side Encryption: The user selects a document in their browser. A React-based frontend encrypts the document locally using AES-GCM before uploading it to Walrus. The raw unencrypted data never leaves the user's browser.
  • NFT Access passes: The encryption key is encrypted with the document owner's public key, and metadata is registered on-chain via a Sui Move smart contract. To share access, the owner mints a dynamic 'AccessPass' NFT and grants it to another wallet. The smart contract validates this relationship, allowing the authorized wallet to decrypt and read the blob on their dashboard.
  • AI Co-Author: An integrated local agent assists with organizing documents, creating semantic indexing (Vector Embeddings) locally inside the browser's IndexedDB, ensuring complete data privacy.

SuiStream AI — Continuous Work-Gated Payments

Estimated Score: 85/100

SuiStream AI addresses the massive friction in freelance and gig-economy payouts. Instead of waiting for monthly invoices, workers receive compensation by the second. Streams are locked in smart contracts, and an autonomous AI agent acts as a supervisor, dynamically adjusting stream rates or delivering milestone bonuses by analyzing active work proof (e.g. GitHub commits, server uptime logs, design deliverables).

How it Works

  • Smart Streaming Contract: Allows an employer to deposit SUI or stablecoins (USDC) into a stream object bound to a worker. The worker can withdraw accrued funds progressively at any second.
  • AI Supervisor Oversight: Rather than manually auditing work, an AI agent acts as a smart controller. It reads milestones, reviews code commits or output metrics using standard LLMs, and triggers rate upgrades or unlocks instant bonuses directly on-chain when objectives are satisfied.
  • Auto-hedging Pool: Integrated with DeepBook v3, allowing workers to activate auto-swap parameters. The incoming streamed volatile SUI is immediately swapped for stablecoins to protect purchasing power.
